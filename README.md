# OceanFin

- **`ui/`** → Next.js frontend  
- **`backend/apps/`** → NestJS backend API  

---

## Quick Start (Step-by-Step)

### **Prerequisites**

Before starting, ensure you have:

- **Node.js ≥ 20**
- **npm ≥ 10**
- **Windows PowerShell or CMD** (recommended)

Check your versions:
```bash
node -v
npm -v
```

### **Clone the Repository**
```bash
git clone https://github.com/Tizun71/OceanFin.git
cd OceanFin
```

### **Install Dependencies**
```bash
npm install
```

### **Start the Development Servers**
```bash
npm run dev
```

### Local URLs
- App OceanFin: http://localhost:3000
